# C++ Calculator

A beginner-friendly C++ calculator built while learning C++. This project performs basic mathematical operations through a simple menu-driven interface.

## ✨ Features
- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- ⬜ Square of a Number
- 🔼 Power of a Number (x^y)
- 📋 Menu-driven interface

## 🛠️ Technologies Used
- C++
- Visual Studio Code
- Git & GitHub

## 🚀 How to Run
1. Clone this repository.
2. Open the project in your C++ IDE or Visual Studio Code.
3. Compile the source file.
4. Run the program and choose an operation from the menu.

## 📚 Concepts Used
- Variables
- Input/Output (`cin`/`cout`)
- `switch` Statement
- Arithmetic Operators
- Basic User Interaction
